もうすぐ12月。そろそろ忘年会や新年会の予定が入る頃ではないでしょうか？ redmineのプラグイン「Scheduling Poll」は、複数人での日程調整が簡単にできる便利プラグインです。

忘年会や同窓会はもちろん、普段の予定も redmine にお任せ。 担当者さんの強い味方となるこのサービス、早速使ってみてくださいね。

インストール

まず git clone と migrate してインストールしましょう！

{% highlight console %} $ cd plugins $ git clone https://github.com/cat-in-136/redmine_scheduling_poll.git $ cd .. $ bundle exec rake redmine:plugins:migrate RAILS_ENV=production {% endhighlight %}

管理者さんは、プロジェクトに対して「Scheduling polls」を有効にしましょう。

スケジュール調整に招待できるユーザーは、プロジェクトのロールで許可されているユーザーが対象です。 *redmineの[管理]＞[ロール]＞[権限レポート]を選択して、「View scheduling polls」と「Vote scheduling polls」の権限の設定をする必要があります。

イベント作成

予定の取りまとめを任されたら、redmine のプロジェクトから[新しいチケット]を選択して、"イベント"のチケットをつくりましょう。

チケットの[Scheduling Poll]＞[Add]を選択して、みんなの予定を聞く"Scheduling poll"ページをつくってみましょう！

プロジェクトの活動には、スケジュール調整を追加した通知が届きます。回答期限やお願いはチケットの注記に添えてくださいね♪

予定の回答

回答だって簡単。チケットの[Scheduling polls]＞[Vote]をタップして、回答画面に移動しましょう。選択肢を選ぶだけで、時間をかけずにかんたんに回答！コメントを残すことだってできます。 *選択肢はプラグインの設定から変更可能です

回答されたイベントは、回答結果がわかりやすく表示されるので、日程決定がスムーズに。Vote をタップすると、下の方にみんなの回答を一覧にしてみることができます。回答は後から変更することもできるので、他に予定が入ってしまった時も安心！

コメントは、チケットの注記として表示されます。参加可能な時間や、持ち物を書き込むにも便利！

作成したイベントの編集や日程の変更の追加はロールで許可したユーザーなら誰でも可能。

これからの季節に大活躍すること間違いなしの「Scheduling polls」。楽しい予定の調整に、お役立てください！
